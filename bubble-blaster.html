Bubble blasters

About this game

Bubble Blasters is a game that i made with a team in the global game jam 2025. We had 29.5 hours to make this game and it was out first one where we needed to make everything ourselves. The goal of the game is to clean the houses with your bubble guns. The game isnt big but that wasnt the point. This is the best game that i have made with almost no help of the teachers.
